# Nextwork

大厂岗位偏好、学生技能勋章墙与高考选专业建议原型。

## 产品思路

这个小程序不是从“专业名称”正推就业，而是倒过来：

1. 抓取大厂官网岗位，拆解岗位类别、业务方向、技能要求和基础能力。
2. 拆解当前高热创业赛道，看这些赛道最终需要什么类型的人。
3. 把岗位需求和创业赛道映射到高考专业群。
4. 让高考生选择自己的学科优势、兴趣和风险偏好。
5. 输出专业推荐、四年学习路线、就业目标和创业验证方向。

## 已有功能

- 大厂岗位数据统一成 `Job` JSON 模型
- 岗位市场洞察、技能匹配和个人勋章墙
- 高考生画像选择
- 就业岗位终局信号
- 创业赛道拆解
- 专业群推荐
- 大学四年学习路线
- 政策与专业目录来源链接

## 运行

```bash
npm install
npm run dev
```

## 数据与分析命令

```bash
npm run scrape:jobs
npm run analyze:jobs
```

当前 `scrape:jobs` 输出种子数据到 `data/jobs.seed.json`。真实上线时建议为每家招聘官网单独写 adapter，负责分页、详情页抽取、去重、增量更新和失败重试。

`analyze:jobs` 输出 `data/analysis.json`，结构适合替换为 ClaudeCode 或 Claude API 生成的标签与分析结果。

## 后续真实接入建议

- 抓取层：每家公司一个 adapter，只保存公开招聘页数据，保留 `sourceUrl` 和抓取时间。
- 标签层：ClaudeCode 将岗位描述抽为分类、技能、业务方向、基础能力、稀缺能力和学习建议。
- 赛道层：维护 AI Agent、具身智能、低空经济、AI 医疗、新能源、AIGC 内容等方向，并定期用政策、融资、招聘和真实产品数据更新热度。
- 评估层：把高考生画像、专业群、岗位能力和创业赛道做可解释匹配。
- 呈现层：小程序首页优先展示“选专业建议”，技能勋章墙作为大学生和求职阶段的延伸功能。
